# 建设路线图

## 1. 可运行底座

- Markdown 内容结构与搜索
- 可复用交互组件
- GitHub Pages 自动构建和发布
- 构建与基础链接检查

## 2. 内容标准

- 分类法、标签、术语表和文章模板
- 数据口径、引用和修订规范
- 技术审核和内容质量清单

## 3. 核心知识

初步候选主题：

- 能量、功率、效率与容量
- 电力系统的发、输、配、用
- 煤炭、石油与天然气
- 光伏、风电、水电、核能与生物质
- 电化学储能、抽水蓄能与热储能
- 能源经济、政策、安全与排放核算

## 4. 运营与更新

- 定期检查失效链接和过期数据
- 记录页面负责人、审核人和最近复核日期
- 使用 issue 收集纠错、内容请求和数据更新

## 待决策

1. 主要读者是公众、学生、从业者还是研究人员？
2. 首批要完成的 3–5 个主题是什么？
3. 中文内容和代码分别采用什么许可证？
4. 仓库名、GitHub 归属与是否使用自定义域名？
